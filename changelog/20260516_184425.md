## Architecture: how this was built

按 `/fanout` playbook 的"先打底、再扇出"模式:

```
Phase 1 (单线契约 ~1h)
  ├─ design tokens / base / layout 三层 CSS
  ├─ Shell 重构成 .layout grid + Welcome empty state
  ├─ 6 个组件 + 7 个 CSS 文件主权清晰化
  └─ commit 0e731dd

Phase 2 (3 agent 并行 ~3h)
  ├─ Track A → commit 8fa9c88 → merge 9cec26d
  ├─ Track B → commit 5940500 → merge 8295c88
  └─ Track C → commit 0027f2f → merge e21ecc3
```

主权设计严格隔离:每 track 只允许动 2-4 个文件,3 个 agent 的 diff 没有任何文件重叠,merge 全部 `--no-ff` 0 冲突。
