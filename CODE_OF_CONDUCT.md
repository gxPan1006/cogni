# Code of Conduct

## Our pledge

We want participation in the Cogni project to be a welcoming, harassment-free
experience for everyone, regardless of background or identity. We pledge to act
in ways that contribute to an open, friendly, and healthy community.

## Our standard

This project adopts the **Contributor Covenant, version 2.1**, as its code of
conduct. The full text — including the expected behaviors, unacceptable
behaviors, enforcement responsibilities, and enforcement guidelines — is the
canonical standard and is available here:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: be respectful, assume good faith, give and accept constructive
feedback gracefully, and focus on what is best for the community. Behavior that
harasses, demeans, or excludes others is not tolerated.

## Scope

This Code applies within all project spaces (the repository, issues, pull
requests, discussions) and when an individual is officially representing the
project in public spaces.

## Reporting

If you experience or witness unacceptable behavior, please report it to the
maintainer. Until a dedicated contact address is published, reach out privately
via the maintainer's GitHub profile ([@gxPan1006](https://github.com/gxPan1006)).
All reports will be reviewed and investigated promptly and fairly, and the
reporter's privacy and security will be respected.
